# Contributing

1. `pnpm install` && `pnpm run build`
2. `pnpm run typecheck` && `pnpm run test` before PRs
3. API changes: sync Pro OpenAPI → `pnpm run contract:sync` → update `contracts/extension-api-surface.json`
4. Internal planning docs go in `docs-internal/` (not pushed)

Issues: https://github.com/ladaojeifang/AssetVault_Browser_Extension/issues
